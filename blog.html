Assignment 1 Entry (10/9/2020):

Assignment 1 was mostly review for me. I took a intro html course in high school so I knew the basics about the tags and how to structure the webpage. The review was useful, and it was more technical than how I learned it in high school. Git was mostly new to me, but I found the lecture on it to be enough to get the hang of it, especially considering I started to look into it a bit over the summer as I was working on some personal projects.

Because I happenned to know a lot of the stuff we talked about, there were not very many challanges and any questions I had were answered by the in class lectures. Therefore, I have yet to make significant use of the class resources (outside of lectures). I don't anticipate this being much of a problem because I have used Piazza and other coding resources in my previous two CS courses and found them to be very useful.

I am excited to start to dive into CSS and especially scripting. CSS was taught breiefly in my high school class but I have already learned things from the first lecture about it here that were not taught at all in my other class. I know CSS is a powerful tool and I am excited to dive deeper into it and actually make webpages that look more professional than I was able to make before.
